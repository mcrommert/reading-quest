Set this up as a kitchen board

One-time iPad config — leaves it always-on, full-screen, auto-refreshing
  1. Open this URL in Safari on the iPad — http://<your-server>:8602/board/ipad.
  2. Tap the Share icon → Add to Home Screen. It launches full-screen with no browser chrome.
  3. Settings → Display & Brightness → Auto-Lock → Never. (Plug the iPad in.)
  4. Settings → Accessibility → Guided Access → On. Triple-click the side button to lock the iPad to this app.
  5. The board auto-refreshes every 60 seconds. Log with /read in Mattermost and it shows up within a minute.
